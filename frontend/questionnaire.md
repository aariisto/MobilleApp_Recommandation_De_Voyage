# Formulaire de préférences de voyage

Ce document définit les questions du formulaire, les réponses possibles et les catégories `categories_simple` associées.

---

## Q1 – Type principal de lieu recherché
**Question :** Quel type de lieu t’intéresse le plus pour cette recherche ?

- A. Lieux touristiques / monuments  
  - Catégories : `tourism`, `tourism.attraction`, `tourism.sights`
- B. Restaurants / bars / cafés  
  - Catégories : `catering`, `catering.restaurant`, `catering.cafe`, `catering.bar`, `catering.pub`
- C. Parcs, nature, plages  
  - Catégories : `leisure`, `leisure.park`, `leisure.park.nature_reserve`, `natural`, `natural.protected_area`, `beach`, `beach.beach_resort`, `national_park`
- D. Shopping / centres commerciaux  
  - Catégories : `commercial`, `commercial.shopping_mall`, `commercial.marketplace`, `commercial.supermarket`, `commercial.gift_and_souvenir`
- E. Vie nocturne  
  - Catégories : `adult`, `adult.nightclub`, `catering.bar`, `catering.pub`
- F. Sport / activités sportives  
  - Catégories : `sport`, `sport.sports_centre`, `sport.stadium`, `sport.swimming_pool`, `sport.track`, `ski`, `ski.lift`, `ski.lift.gondola`
- G. Musées / culture  
  - Catégories : `entertainment`, `entertainment.museum`, `entertainment.culture`, `entertainment.culture.arts_centre`, `entertainment.culture.theatre`
- H. Lieux religieux / spirituels  
  - Catégories : `religion`, `religion.place_of_worship`, `tourism.sights.place_of_worship`

## Q2 – Type précis de restauration
**Question :** Quel type de restaurant préfères-tu ?

- A. Cuisine locale / régionale  
  - Catégories : `catering.restaurant.regional`, `catering.restaurant.mediterranean`, `catering.restaurant.spanish`, `catering.restaurant.french`, `catering.restaurant.portuguese`, `catering.restaurant.moroccan`
- B. Cuisine italienne (pizza, pâtes, etc.)  
  - Catégories : `catering.restaurant.italian`, `catering.restaurant.pizza`
- C. Cuisine asiatique (japonaise, chinoise, thaï, vietnamienne…)  
  - Catégories : `catering.restaurant.japanese`, `catering.restaurant.sushi`, `catering.restaurant.chinese`, `catering.restaurant.thai`, `catering.restaurant.vietnamese`, `catering.restaurant.korean`, `catering.restaurant.asian`, `catering.restaurant.ramen`, `catering.restaurant.noodle`
- D. Cuisine américaine (burgers, steak house…)  
  - Catégories : `catering.restaurant.american`, `catering.restaurant.burger`, `catering.restaurant.steak_house`
- E. Cuisine végétarienne / vegan  
  - Catégories : `vegetarian`, `vegetarian.only`, `vegan`, `vegan.only`
- F. Aucune préférence  
  - Catégories : `catering.restaurant`
